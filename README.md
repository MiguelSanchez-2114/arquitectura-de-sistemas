
# Arquitectura de Sistemas para IA

Autor: Miguel Sánchez

Curso: Arquitectura de Sistemas para Inteligencia Artificial

Este repositorio recoge materiales y notas del curso "Arquitectura de Sistemas para IA". El objetivo del curso es comprender cómo diseñar, desplegar y mantener sistemas que soportan aplicaciones de inteligencia artificial, incluyendo infraestructura, canalizaciones de datos, entrenamiento y despliegue de modelos, escalabilidad, monitorización, seguridad y consideraciones éticas.
